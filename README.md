# VN Dev — Portfólio & Orçamentos

Landing page interativa para solicitação de orçamentos de soluções Full Stack.

## ✨ Features

- 🤖 Robô 3D interativo (Spline) — segue o mouse em tela cheia
- 📋 Formulário de orçamento com seleção de serviços
- 🎞️ Ticker de projetos entregues com scroll animado
- 🌙 Design dark futurístico com efeito spotlight no mouse
- 📱 Responsivo para mobile

## 🛠️ Stack

- HTML5 + CSS3 puro (sem frameworks)
- [Spline 3D Viewer](https://spline.design/) para o robô interativo
- Google Fonts: Space Grotesk + JetBrains Mono

## 🚀 Deploy

Hospedado no [Vercel](https://vercel.com/).
